# 📊 Customer Retention & Churn Analysis Dashboard

##  Project Overview

This project presents an interactive **Customer Retention & Churn Analysis Dashboard** developed in Microsoft Excel using the IBM Telco Customer Churn dataset. The dashboard helps analyze customer behavior, identify churn patterns, evaluate retention drivers, and provide actionable business insights through interactive visualizations.

---

##  Objective

The objective of this project is to analyze customer churn and retention trends for a subscription-based business. The dashboard enables decision-makers to identify high-risk customer segments and implement strategies to improve customer retention.

---

##  Dashboard Features

- 👥 Total Customers
- ✅ Active Customers
- ❌ Churn Customers

- 📅 Average Customer Tenure
- 💰 Average Monthly Charges
- 📄 Churn by Contract Type
- 🌐 Churn by Internet Service
- 👤 Churn by Gender
- 💳 Payment Method Analysis
- 🎛️ Interactive Slicers

---

##  Key Insights

- Overall customer churn rate is **26.54%**.
- Month-to-Month contract customers have the highest churn.
- Fiber Optic customers show a higher churn rate than DSL users.
- Customers using Electronic Check have higher churn compared to other payment methods.
- Long-term customers demonstrate better retention.
- Higher monthly charges are associated with increased customer churn.

---

## Business Recommendations

- Encourage customers to choose long-term contracts.
- Promote automatic payment methods to improve retention.
- Improve onboarding for new customers.
- Provide targeted offers for high-risk customer segments.
- Review pricing strategies for customers with high monthly charges.
